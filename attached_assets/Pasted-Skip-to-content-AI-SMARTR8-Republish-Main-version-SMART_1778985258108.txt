LARGE BUILD: Fix worksheet bugs + make every main-page button 
launch a unified funnel. Read all sections before starting.

============================================
PART 1: FIX EXISTING WORKSHEET BUGS FIRST
============================================

BUG 1 — Stuck "0" placeholder in number inputs:
Every number input across the entire site (worksheet, all 
funnels, all forms) currently shows a pre-filled "0" that the 
user cannot delete. When they type, the 0 stays.

Fix: number inputs must be EMPTY by default with a gray 
placeholder (e.g. placeholder="350000"). The placeholder 
disappears when the user types. No pre-filled "0" values 
anywhere. Audit every <input type="number"> site-wide.

For fields with smart defaults (like the 5.125% rate or 
$3,999 closing costs in step 4), those defaults should be 
actual editable values the user can fully clear and retype, 
NOT stuck placeholders.

BUG 2 — Results screen shows full worksheet before lead 
capture:
On the public worksheet results screen, the full worksheet 
PDF preview is currently visible. We want the value teased 
but not fully readable.

Fix: on the results screen, show the THREE big summary 
numbers fully visible and clear (Monthly Savings, Interest 
Saved, Years Shaved Off). Below that, show the detailed 
worksheet preview with a blur/frosted-glass overlay on top 
of it — the user can see there's a full document there, but 
can't read the details. Over the blurred area, a centered 
message: "Your full worksheet is ready. Enter your info to 
get the complete breakdown emailed to you." with the "Email 
My Worksheet" button.

BUG 3 — Emailed PDF layout is broken/funky:
The current PDF has cramped columns, awkward text wrapping, 
and poor spacing (see the narrow squished tables). 

Fix: rebuild the PDF layout to be clean and professional:
- Full-width tables that use the available page width
- Proper padding and spacing between sections
- Text never squished into narrow columns
- The "How This Strategy Works" section should be a normal 
  readable paragraph block, NOT a narrow sidebar column
- Letter size, proper margins (0.5in)
- Clean page breaks (page-break-inside: avoid on tables, 
  banner, signature, compliance footer)
- It should look like a polished professional one-to-two 
  page document

============================================
PART 2: UNIFIED FUNNEL FOR ALL MAIN-PAGE BUTTONS
============================================

Every product button on the smartr8.com main page must 
launch the SAME unified funnel. Buttons covered:
- Cash-Out Refinance
- Rate Reduction Refinance
- Home Equity / 2nd Mortgage
- HELOC
- Purchase

The button the user clicks pre-sets the "product type" 
context but the funnel structure is identical for all.

FUNNEL = the same multi-step guided flow already built for 
the worksheet (4-step style: product, current situation, 
goals, new loan details). Required fields stay required, 
optional fields stay optional. Reuse the worksheet funnel 
components — do NOT build separate funnels per button. One 
shared funnel component, product type passed in as a prop.

GLOBAL FUNNEL UI (every step):
- Progress indicator at top
- Persistent "Start Over" button (clears sessionStorage, 
  returns to step 1) — on EVERY step and the results screen
- Back button (except step 1)
- Mobile-first, each step fits one screen where possible
- Capture every answer into sessionStorage as the user 
  progresses

============================================
PART 3: THE FORK SCREEN (CRITICAL NEW FEATURE)
============================================

After the user completes the essential funnel questions, 
BEFORE they reach the /whats-next page, show a fork screen.

Fork screen header: "How would you like your quote?"
Subtitle: "Both options are 100% free — no cost either way."

TWO large choice cards:

CARD 1 — "Create My Own Quote"
- Icon: calculator or sliders
- Headline: "Build It Yourself"
- Body: "Take the wheel. Enter your own numbers, add your 
  debts, and instantly see your personalized savings and 
  payoff strategy. Perfect if you like to explore the 
  numbers on your own."
- Button: "Create My Quote"

CARD 2 — "Have a Licensed Professional Build It"
- Icon: person/handshake
- Headline: "Done For You"
- Body: "Let Mykoal, a licensed loan officer, build your 
  personalized quote and walk you through your best options. 
  Perfect if you'd rather have an expert handle the details."
- Button: "Get My Professional Quote"

Both cards clearly display: "No cost. No obligation. No 
credit pull."

============================================
PART 4: FORK PATH LOGIC
============================================

  payload. Source: "smartr8-funnel". Include which button

- This happens ONCE, on funnel completion, before the fork
