```
GOOGLE PLACES API SETUP — Step by Step

1. Go to https://console.cloud.google.com
2. Sign in with your Google account
3. Top of page, click the project dropdown, then click "New Project"
4. Project name: Smartr8 Forms
5. Click Create. Wait for it to provision (15-30 seconds)
6. Make sure the new project is selected in the top dropdown
7. Left sidebar: Click "APIs & Services" then "Library"
8. Search "Places API" — click it — click ENABLE
9. Go back to Library
10. Search "Maps JavaScript API" — click it — click ENABLE
11. Left sidebar: APIs & Services → Credentials
12. Click "+ CREATE CREDENTIALS" at top → "API key"
13. A popup shows your API key. COPY IT.
14. Click "Edit API key" (pencil icon) on the new key
15. Under "Application restrictions":
    - Select "HTTP referrers (web sites)"
    - Click "ADD AN ITEM" and add each of these:
      - https://smartr8.com/*
      - https://www.smartr8.com/*
      - https://*.pages.dev/*  (for Cloudflare preview deployments)
      - http://localhost:*  (for local development)
16. Under "API restrictions":
    - Select "Restrict key"
    - Check: Places API AND Maps JavaScript API
17. Click Save

18. In Replit, add the API key as an environment variable:
    - Open the "Secrets" tab in your Replit
    - Key: GOOGLE_PLACES_API_KEY
    - Value: paste your API key
    - Save

19. Set up billing (required even for free tier):
    - In Google Cloud Console, go to "Billing"
    - Add a payment method
    - Google gives you $200/month free credit which covers ~10,000 address lookups
    - You won't be charged unless you exceed that (you won't)

20. Verify: Send Replit a test, "Check if Google Places autocomplete is working on the test page" — if you see address suggestions appear when typing, it's working.
```

The Replit code itself should:
- Use the Google Maps JavaScript API loader (do NOT install heavy npm packages like @react-google-maps/api — use the lightweight @googlemaps/js-api-loader or vanilla JS)
- Reference the API key via process.env.GOOGLE_PLACES_API_KEY (or framework equivalent)
- Show a graceful fallback if API key is missing: a regular text input with a note "Address autocomplete temporarily unavailable, please type your full address"
- Only load the Google Maps script on funnel pages where it's needed (not the homepage) — performance optimization
